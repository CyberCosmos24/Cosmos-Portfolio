# Cosmos Portfolio

A refreshed, professional portfolio for Ajay Kalra focused on security engineering, projects, and ways to connect. The site highlights hands-on security work, provides quick links to case studies, and keeps a clean, accessible layout for sharing with potential collaborators and employers.

## Pages
- **Home**: Intro, highlights, skills, and contact links.
- **Projects**: Overview cards with impact, skills, and links to detailed case studies.
- **Case studies**: Deep dives for Drexel CCI Security Toolkit and Veilock (formerly AstroVPN).
